Visualizador de Script Python desde el Servidor Rocks

Este documento HTML cargará y mostrará el contenido de un script Python directamente desde tu nodo maestro de Rocks.

Configuración Necesaria en tu Nodo Maestro de Rocks:

  • Asegúrate de que un servidor web simple (como python3 -m http.server 8000) esté ejecutándose en tu nodo maestro de Rocks (Front End).
  • El archivo de tu script Python (ej., mi_script.py) debe estar en el directorio donde iniciaste el servidor HTTP.
  • El puerto 8000 (o el que uses) debe estar abierto en el firewall de tu nodo maestro de Rocks.
  • La dirección IP de tu nodo maestro de Rocks (Front End) para acceder desde el host es 10.0.3.15[cite: 15].

Si la configuración es correcta, el contenido de tu script aparecerá a continuación.
